# 🎬 Netflix Sentiment Analysis

## 📌 Overview

This project performs sentiment analysis on Netflix content descriptions using Natural Language Processing (NLP). It classifies each title as Positive, Neutral, or Negative and analyzes trends over time.

---

## 🎯 Objective

* Analyze Netflix dataset
* Perform sentiment classification using TextBlob
* Visualize trends based on release year
* Extract meaningful insights from data

---

## 🛠️ Tech Stack

* Python
* Pandas
* TextBlob
* Plotly

---

## 📊 Features

* ✔️ Data cleaning and preprocessing
* ✔️ Sentiment classification (Positive / Neutral / Negative)
* ✔️ Year-wise sentiment distribution
* ✔️ Interactive visualization

---

## 📈 Key Insights

* Most Netflix content has **positive sentiment**
* Content increased significantly after **2015**
* Negative sentiment is comparatively **low**
